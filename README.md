# Hr-Analytics-Dashboard
This project focuses on analyzing employee data to identify attrition trends, workforce distribution, and key HR performance metrics.  The objective is to build an interactive dashboard that helps HR teams make data-driven decisions related to employee retention, salary structure, and job satisfaction.
Problem Statement

Organizations often struggle to track employee attrition, departmental performance, and workforce demographics due to scattered reporting systems.

This project aims to:

Monitor employee attrition rate

Analyze department-wise employee distribution

Identify factors influencing employee turnover

Track salary, age, and job satisfaction metrics
